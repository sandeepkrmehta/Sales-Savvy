## Sales Savvy 🛍️

A full-stack e-commerce application built with **Spring Boot**, **React.js**, and **MySQL**. Features include user authentication, product browsing, shopping cart, and role-based access control.

### 🔧 Tech Stack
- Backend: Spring Boot, Spring Security, JPA, MySQL
- Frontend: React.js, JavaScript, Tailwind CSS
- Tools: Postman, Git, VS Code

### ✅ Features
- RESTful APIs for products, users, orders
- Secure login and JWT-based role management
- Persistent shopping cart and order history
- Mobile-responsive design
